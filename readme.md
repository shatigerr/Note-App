# MERN STACK NOTE APP

## Technologies
* MongoDB
* ExpressJS
* REACT
* NodeJS

## How to initialize the project

### clone the repository
```git clone https://github.com/shatigerr/Note-App.git```

### Install dependencies

* ```cd Note-App/ ```
* ```cd server/```
* ```npm install ```
* ```cd frontend/```
* ```npm install ```

### Run the APP

* ```npm run dev```
* ```cd ./server/```
* ```npm run dev`
